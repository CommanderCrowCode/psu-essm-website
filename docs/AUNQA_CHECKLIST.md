# AUN-QA Completeness Checklist — ESSM Website

Source: aunqa-lead (relay-mesh), grounded in the **AUN-QA V4 guide (Oct 2020 ed.)** +
the program's **2024 SAR**. Use this as the **step-5 QA-committee checklist** and as the
content-gap guide for the populate-crew.

## Framing (important)

AUN-QA V4 is a **self-assessment (SAR) framework**, *not* a website-content spec — there is
**no formal "required public-facing website checklist."** We do not invent one. But the site
is genuinely assessment-relevant:

- The V4 guide lists **"University and faculty websites" as a source of evidence** for
  **Criterion 1 (Expected Learning Outcomes)** and **Criterion 2 (Programme Structure)**.
- V4 diagnostic questions for C2 ask outright: *"Are the programme specifications published
  and made available to stakeholders?"* and *"Are the programme and courses specifications
  published online?"*

So assessors cite the site directly. The bar: **comprehensive, up-to-date, made available +
communicated to all stakeholders.**

## Mandatory-to-publish core

### Programme specification (must contain — 8 fields)
- [ ] Awarding body/institution (+ teaching institution if different)
- [ ] Details of accreditation by professional/statutory bodies
- [ ] Name of final award + programme title
- [ ] Expected Learning Outcomes (programme-level / PLOs)
- [ ] Admission criteria / requirements
- [ ] Relevant benchmark / reference points informing the ELOs
- [ ] Programme structure & requirements (levels, courses, credits)
- [ ] Date of writing

### Course specification (per course)
- [ ] Title; requirements / pre-reqs / credits
- [ ] Course-level ELOs (knowledge / skills / attitude)
- [ ] Teaching-learning-assessment methods aligned to ELOs (constructive alignment)
- [ ] Description / outline / syllabus
- [ ] Assessment details
- [ ] Date written / revised

## Criterion → Page map

| AUN-QA Criterion | Page / section |
|---|---|
| C1 Expected Learning Outcomes | Curriculum (ELO list) + About (aims / stakeholder alignment) |
| C2 Programme Structure & Content | Curriculum (structure, courses, credits, course specs) + Admissions (benchmarks) |
| C3 Teaching & Learning Approach | Curriculum / About (educational philosophy, T&L methods, constructive alignment) |
| C4 Student Assessment | Curriculum / Admissions (assessment methods + published grading criteria; thesis assessment) |
| C5 Academic Staff | People (faculty profiles, qualifications, research areas) |
| C6 Student Support Services | Admissions / Contact / About (advising, supervision model, student facilities) |
| C7 Facilities & Infrastructure | Research / About (labs, field stations, IT, library) — section, not mandatory page |
| C8 Output & Outcomes | Research (publications, graduate outcomes) + QA/accreditation statement |

## QA-committee binary checklist (AUN-QA-completeness)

- [ ] Programme-level ELOs visible
- [ ] Programme spec — all 8 fields present
- [ ] Course specs published online w/ course-ELOs + credits + assessment
- [ ] Admission criteria explicit
- [ ] Assessment + published grading criteria
- [ ] Academic-staff profiles
- [ ] Student-support / supervision described
- [ ] QA / accreditation statement
- [ ] **All facts source-gated (no guesses)** — hold this line
