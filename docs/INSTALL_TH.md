# วิธีติดตั้งเว็บไซต์หลักสูตร ESSM เป็นโฟลเดอร์ย่อยของเว็บไซต์ WordPress

เว็บไซต์นี้เป็นไฟล์ static (HTML / CSS / รูปภาพ) ทั้งหมด ไม่ต้องใช้ฐานข้อมูล ปลั๊กอิน หรือธีมของ WordPress เพียงนำโฟลเดอร์ไปวางในเว็บไซต์เดิม แล้วเปิดใช้งานเป็นที่อยู่ย่อย (เช่น `te.psu.ac.th/essm`) ได้ทันที

## สิ่งที่ต้องเตรียม
- ไฟล์ `essm-website.zip` (ส่งมาพร้อมกัน)
- สิทธิ์เข้าถึงพื้นที่เก็บไฟล์ของเว็บไซต์ ผ่าน FTP/SFTP (เช่น โปรแกรม FileZilla) หรือ File Manager ใน cPanel / DirectAdmin

## ขั้นตอนการติดตั้ง
1. **แตกไฟล์ ZIP** — แตกไฟล์ `essm-website.zip` ออกมา จะได้ไฟล์และโฟลเดอร์ของเว็บไซต์ ได้แก่ `index.html` และโฟลเดอร์ `about/`, `curriculum/`, `admissions/`, `research/`, `contact/`, `assets/` (รวมถึงไฟล์อื่น ๆ เช่น `sitemap.xml`)

2. **รวมไว้ในโฟลเดอร์ชื่อ `essm`** — นำไฟล์ทั้งหมดที่แตกออกมาไว้ในโฟลเดอร์เดียวชื่อ `essm` โดยชื่อโฟลเดอร์นี้จะกลายเป็นที่อยู่ของเว็บไซต์ คือ `/essm`

3. **อัปโหลดขึ้นเซิร์ฟเวอร์** — อัปโหลดโฟลเดอร์ `essm` ทั้งโฟลเดอร์ ไปไว้ใน “โฟลเดอร์หลัก” ของเว็บไซต์ ซึ่งเป็นที่เดียวกับที่ติดตั้ง WordPress (โดยทั่วไปคือ `public_html` หรือ `www`)
   - เมื่ออัปโหลดเสร็จ เส้นทางไฟล์ควรเป็น `public_html/essm/index.html`

4. **ตรวจสอบว่าใช้งานได้** — เปิดเว็บเบราว์เซอร์ ไปที่ `https://te.psu.ac.th/essm/` ควรเห็นหน้าแรกของเว็บไซต์ ESSM พร้อมรูปภาพและเมนูด้านบน จากนั้นลองคลิกเมนู About / Curriculum / Admissions / Research / Contact เพื่อตรวจว่าทุกหน้าเปิดได้ตามปกติ

5. **การตั้งค่าเพิ่มเติม** — โดยปกติ **ไม่ต้องตั้งค่าใด ๆ เพิ่มเติม** เพราะเป็นเว็บไซต์ static ที่ไม่พึ่งพาฐานข้อมูลหรือปลั๊กอิน

## หากพบปัญหา
- **หน้าเว็บเปิดได้ แต่หน้าตา/รูปภาพไม่แสดง (เหมือนไม่มีดีไซน์):** ให้ตรวจว่าได้อัปโหลดโฟลเดอร์ `assets/` ครบถ้วน และอยู่ภายในโฟลเดอร์ `essm/`
- **เปิด `/essm` แล้วขึ้นหน้า “ไม่พบหน้า” (404) ของ WordPress:** โดยปกติ WordPress จะข้ามคำขอที่ตรงกับไฟล์/โฟลเดอร์ที่มีอยู่จริง จึงไม่รบกวนโฟลเดอร์ `essm` หากยังพบปัญหา ขอให้ผู้ดูแลเซิร์ฟเวอร์ตรวจสอบว่ากฎในไฟล์ `.htaccess` ของ WordPress ยังคงข้ามไฟล์/โฟลเดอร์จริง (ซึ่งเป็นค่ามาตรฐานอยู่แล้ว)
- ลิงก์ภายในและที่อยู่รูปภาพทั้งหมดเป็นแบบ relative จึงทำงานได้ทันทีเมื่อวางในโฟลเดอร์ `essm` โดยไม่ต้องแก้ไขไฟล์ใด ๆ
